FaviGrab is now available on the [Chrome Web Store](https://chromewebstore.google.com/detail/favigrab-favicon-grabber/cnpgabmfnfehdamobkafalnpdoigdlil).

## What's Included

- **One-click favicon download** from any website
- **Multiple sizes** — 16x16, 32x32, 48x48, and 64x64
- **PNG and ICO export** for web and desktop use
- **Auto-named files** using the source website's domain
- **Manifest V3** built for modern Chrome

## Why We Built This

Grabbing favicons shouldn't require inspecting page source or guessing URL patterns. FaviGrab makes it a single click. We built it because we needed it ourselves, and figured others might too.

## What's Next

We're exploring SVG favicon detection and batch downloading for power users. If you have ideas, open an issue on [GitHub](https://github.com/blossomstudios).

Install FaviGrab free from the [Chrome Web Store](https://chromewebstore.google.com/detail/favigrab-favicon-grabber/cnpgabmfnfehdamobkafalnpdoigdlil).
